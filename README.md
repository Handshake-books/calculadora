# Calculadora proyecto editorial

Simula la viabilidad económica de una publicación independiente.

Esta herramienta permite estimar los costes, ingresos y reparto económico de un proyecto editorial de forma sencilla. Está pensada para editoriales independientes, autoras/es que se autoeditan y estudiantes de edición.

La calculadora funciona completamente en el navegador y no requiere servidor.

---

---

## Qué permite calcular

- Costes de producción del libro
- Tirada y stock fuera de venta
- Coste unitario
- PVP y desglose de IVA
- Beneficio por unidad
- Punto de equilibrio (ventas necesarias para recuperar la inversión)
- Reparto de inversión entre actores
- Reparto de ingresos
- Resultados económicos por actor
- Exportación a PDF
- Compartir proyectos mediante enlace

---

## Cómo funciona

Todos los cálculos se realizan en el navegador mediante JavaScript.  
No se almacenan datos en ningún servidor.

Cuando se usa la función **Compartir proyecto**, los datos se codifican en la URL para que otra persona pueda abrir exactamente el mismo escenario económico.
